# 日常活动管理工具

一个基于 React + Vite + Ant Design 的日常活动管理网页应用。

## 技术栈

- **Vite** - 快速的前端构建工具
- **React** - 用户界面库
- **Ant Design** - 企业级 UI 组件库
- **Lucide-react** - 现代化图标库
- **React Router** - 路由管理

## 项目结构

```
src/
├── components/      # 公共组件
├── pages/          # 页面组件
├── layouts/        # 布局组件
├── styles/         # 样式文件
├── utils/          # 工具函数
├── hooks/          # 自定义 Hooks
├── services/       # API 服务
└── constants/      # 常量定义
```

## 功能模块

- 📊 仪表盘 - 查看活动统计和进度
- 📅 活动管理 - 管理日常活动
- ✅ 任务列表 - 任务跟踪和管理
- 📈 统计分析 - 数据分析和报表
- 👥 用户管理 - 用户信息管理
- ⚙️ 系统设置 - 系统配置

## 开始使用

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 开发说明

项目已配置好 Ant Design 的基础样式，并创建了完整的项目结构。路由使用 React Router 进行管理，主布局包含侧边栏导航和顶部栏。
